# 项目概述：
图宝是一个基于 AI 的小红书图文生成器，旨在让内容创作变得简单高效。通过输入一句话，系统就能自动生成完整的小红书图文内容，包括文案大纲和配套图片。

# 核心功能
红墨通过 AI 技术简化内容创作流程，主要提供以下核心功能：
功能模块	描述	
智能大纲生成	基于用户输入自动生成 6-9 页的小红书内容大纲	Gemini 3 API 
批量图片生成	并发生成所有页面的配图，支持小红书标准比例	🍌Nano banana Pro 等多服务商 
实时进度追踪	通过 SSE 流式传输实时显示生成进度	Flask SSE 实现 
个性化定制	支持上传参考图片，保持品牌视觉风格一致	图片分析和风格匹配
历史管理	保存和管理创作历史，支持重新编辑和下载	本地存储系统 


# 后端架构
后端基于 Flask 框架构建，采用模块化设计：

API 层: 提供 RESTful 接口，支持 CORS 跨域
服务层: 业务逻辑处理，包括大纲生成、图片生成、历史管理
生成器层: 抽象的 AI 服务接口，支持多服务商切换
工具层: 通用工具函数，包括客户端封装、图片处理等
前端架构
前端采用 Vue 3 + TypeScript 技术栈：

组件化: 基于 Vue 3 Composition API 构建可复用组件
状态管理: 使用 Pinia 进行全局状态管理
路由管理: Vue Router 处理页面导航
构建工具: Vite 提供快速的开发和构建体验


# 工作流程：
1. 输入阶段: 用户输入创作主题，可选择上传参考图片以保持风格一致性 
2. 大纲生成: Gemini 3 分析输入内容，生成结构化的大纲
3. 内容确认: 用户可以编辑页面顺序和具体描述，确保内容符合预期
4. 图片生成: 系统并发生成所有页面配图，支持最高 25 个并发任务
5. 实时更新: 通过 SSE 技术实时推送生成进度，用户无需刷新页面
6. 结果交付: 生成完成后提供一键下载功能
